# jainilajmera.com

Personal website built with React, TypeScript, and Vite.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

The site is deployed to GitHub Pages automatically on push to main. The build output in `dist/` is served at [jainilajmera.com](https://jainilajmera.com).
